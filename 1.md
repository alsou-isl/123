## Шаги тестирования
| Шаг |  Действие  |           ОР          | Статус |

| 4 | В поле "Способ отбивки забоя" выбрать значение из списка | Текст красного цвета ниже поля "Способ отбивки забоя" исчезает |  |
| 5 | Очистить поле "Глубина текущего забоя" | Поле становится пустым |  |
| 6 | Нажать на Сохранить | Ниже поля "Глубина текущего забоя" отображается красным цветом текст "Выберите значение из списка". <br>Ниже поля "Дата замера забоя текущего" отображается красным цветом текст "Выберите значение".<br>Ниже поля "Состояние забоя" отображается красным цветом текст "Выберите значение из списка". | / |
